Política de Privacidad

En Alfred Inc nos tomamos muy en serio la privacidad y seguridad de nuestros usuarios. Nuestra aplicación utiliza los datos únicamente para ofrecer las funciones solicitadas y no compartimos ni vendemos información personal.

Información que recolectamos

Recopilamos la información mínima necesaria para brindar el servicio, como nombre y cuenta asociada al canal de YouTube autorizado.

Uso de permisos de Google APIs

Nuestra aplicación utiliza las APIs de Google para ofrecer funcionalidades específicas al usuario:

  • YouTube (youtube.upload): Este permiso se utiliza exclusivamente para subir videos al canal de YouTube del usuario, según las acciones que el usuario solicite.
  • Google Drive (drive.file): Este permiso se utiliza únicamente para almacenar temporalmente archivos de video generados por la aplicación y facilitar su subida a YouTube.

No compartimos ni vendemos los datos del usuario. El acceso está limitado al alcance mínimo necesario para operar estas funciones.

Para más información sobre la política de datos de Google, visite policies.google.com/privacy.
